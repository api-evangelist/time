# TIMEThis is a repo for managing machine readable artifacts for the [TIME APIs](http://ap.org). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [TIME APIs](http://ap.org).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://ap.org)- [Developer](http://ap.org)- [Twitter](https://twitter.com/AP)- [GitHub](https://github.com/TheAssociatedPress)The [APIs.json](https://github.com/api-evangelist/time/blob/master/apis.json) index provides links to all the relevant links for the TIME APIs, including the OpenAPI and Postman if they are present.
